# 우아한 테크코스 프리코스 - 오픈 미션: 프리코스 챌린지

## 프로젝트 명: 오늘의 맛기온
* 기획 의도
저는 배달의 민족과 관련있는 서비스를 기획했습니다. 음식을 선택할 때, 사람들이 가장 많이 겪는 것이 바로 '선택'입니다. 명확하게 선택할 음식이 없는 경우에 항상 무엇을 먹어야 할 지에 대해 깊은 고민에 빠집니다. 저 또한 그러합니다. 그래서 이번 오픈 미션에서 이를 조금이나마 해소시켜줄 `기온 맞춤 AI 음식 추천 서비스`를 기획했습니다.   

현재 사용자의 위치 기반의 날씨, 온도, 체감 온도, 미세먼지 데이터와 사용자가 선택한 기분 값을 활용하고 이를 기반으로 AI에게 알맞는 음식 추천 받습니다.   

추가적으로 로또 미션을 그대로 가져오되 자바스크립트가 아닌 다른 언어로 미니 게임 형식으로 구현할 계획이며, 터미널에서 진행된 로또 게임을 실물화하여 페이지에서 즐길 수 있도록 만들 계획입니다.   

* 기능 구현
- [ ] 피그마로 UI 디자인 기획
- [ ] 사용자의 위치 정보 데이터 받기
- [ ] 사용자의 위치 기반 날씨, 온도, 체감온도, 미세먼지 등 필요한 데이터 저장
- [ ] 다양한 기분 상태를 나타내고, 사용자가 선택한 기분 상태 저장
- [ ] API를 활용해 AI에게 저장한 데이터를 전달해 오늘의 추천 음식 요청
- [ ] API에 JSON 등 바로 사용할 수 있는 값으로 반환 뒤 페이지에 출력 (이미지, 설명 또는 문구 포함)

* 요구 조건
1. 1~3주 차 미션들의 요구사항을 그대로 충족할 것
2. 지금까지 사용해보지 않은 기술 또는 한 두번 따라해보는 정도의 사용으로 거의 경험해보지 못한 기술을 사용할 것
3. 기존에 사용해보지 않았던 외부 API 등 최대한 도전하는 형식으로 진행할 것

* 기술 선택
그동안 `React`, `TypeScript`, `Next.js` 등 자바스크립트 기반으로 자주 사용하는 프론트엔드(+ 백엔드) 기술을 사용했습니다. 그러나, 이번 미션에서는 이들을 제외하고 처음 사용해보거나 거의 사용해본 적 없는 경험이 없는 것과 같은 기술을 선택하는 것을 우선적으로 고려했습니다.   

또한, 프론트엔드 위주의 경험으로 이번에 간단하지만 백엔드까지 구현하는 것을 목표하였습니다.   

그래서 저는 다음과 같은 기술을 선택했습니다.
1. FrontEnd - Vue
2. BackEnd - Python, Rust

* 기타사항
저는 매일 우아한테크코스 디스코드에 접속해 잡담, 문의사항, 공지사항, 리뷰 탭에서 사람들이 작성한 글에서 동기부여를 얻거나 필요한 정보를 얻는 시간을 가졌습니다. 그리고 가능하다면 제가 공유할 수 있는 정보나 동기부여를 줄 수 있는 내용을 찾고자 노력했습니다.
